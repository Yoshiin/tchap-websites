Tchap
Importez vos clés
Pour importer vos clés, vous aurez besoin de deux choses :
- Le fichier de backup de vos clés tchap-keys.txt
- La phrase secrète liée à ce fichier
I. Paramètres
Pour commencer, il vous faudra accéder à vos paramètres de l'application.
1. Cliquez sur votre nom en haut à gauche de l'application
2. Cliquez sur Paramètres

(Voir la version animée)
II. Sécurité & Vie privée
Accédez aux paramètres de sécurité
1. Cliquez sur l'onglet Sécurité & Vie privée

(Voir la version animée)
III. Import
Importez vos clés
1. Cliquez sur Importer les clés de chiffrement de bout en bout

(Voir la version animée)

2. Sélectionez le fichier tchap-keys.txt que vous avez téléchargé lors de l'export de vos clés

(Voir la version animée)

3. Entrez la phrase secrète que vous avez saisie lors de l'export de vos clés

(Voir la version animée)

4. Cliquez sur Importer
